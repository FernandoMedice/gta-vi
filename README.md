# GTA VI - Página de Teste

Este projeto é uma página web estática criada para fins de estudo, utilizando HTML, CSS e JavaScript.

## Sobre o projeto

A página foi desenvolvida com o objetivo de praticar a criação e publicação de sites simples na internet, utilizando GitHub e futuramente uma plataforma de deploy, como Vercel, Netlify ou GitHub Pages.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
